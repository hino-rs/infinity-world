# 気温と湿度について

infinity-worldにおける、気温と湿度の計算についてまとめる。

## 標高の影響

前提として、高さは0mから10000mまで生成され、高さも気温に影響する。

現実の気温減率をもとに、海面から100mあたり0.65度低下する処理を実装する。

例えばある経緯度の標高0m(海面)が23.0℃だとしたら、その経緯度の標高1000m地点は$23.0 - 0.65\times (1000 / 100)$で$16.5$℃となる。

## ノイズによる分布

高さだけだとY軸に単調になってしまうので、ここにもノイズをかける。
一旦、perlinを低オクターブ数のfbmにかけることで作る。

## 湿度

湿度も同じくノイズで作るが、加えて、求めるための飽和水蒸気量は現実と同じく(とはいえ疑似)気温をもとに計算する。

# Infinity World

Rust製ボクセルエンジン。
無限に続く美しい景色を、PCに低負荷で歩き続けられることが目標・

## 📌Policy

- 地下無し
- ワールド保存・キャッシュ無し
- 表面特化

## ✨Features

- ブロック表示
- カメラ・カメラ操作
- 空・太陽
- コヒーレントノイズを用いたチャンク生成
- チャンクメッシュ
- ボクセルAO
- 座標表示
- FPS表示
- 複数のチャンク生成
- AABB
- 浮遊モード

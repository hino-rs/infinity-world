# Infinity World

Rust製ボクセルエンジン。
無限に続く美しい景色を、PCに低負荷で歩き続けられることが目標

## 📌方針

- 地下無し
- ワールド保存無し
- 表面特化
- Wasm特化

## ✨実装済み

- カメラ・カメラ操作
- 空・太陽
- コヒーレントノイズを用いたチャンク生成
- ボクセルAO
- 座標表示
- FPS表示
- 複数のチャンク生成
- プレイヤーAABB
- 浮遊モード
- グリーディメッシュ
